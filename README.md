## ğŸš€ Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd futurise-desktop

# Install dependencies
npm install

# Start development server
npm run dev
```
